KeenKeeper
Project Overview

Friend Tracker is a simple web application that helps you manage and maintain your personal relationships. You can track when you last contacted your friends, set goals, and log interactions like calls, texts, and video chats.

 Features
 View all friends in a clean card layout

 Status indicators (Overdue, Almost Due, On Track)
 Timeline to see all interactions (Call, Text, Video)
 Quick Check-In buttons to log interactions instantly
 Friendship Analytics with Pie Chart
 Fully responsive design


Technologies Used
Next.js
Tailwind CSS + DaisyUI
Recharts (for analytics chart)
JSON (for mock data storage)

Key Functionalities
Add and track friend interactions (Call, Text, Video)
Dynamic timeline updates with toast notifications
Analytics page showing interaction distribution
